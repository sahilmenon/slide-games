# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities by emailing **sahilmenon01@gmail.com** with the subject line `[slide-games] Security Report`. Include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept (if safe to share)
- Any suggested remediation

You will receive an acknowledgement within **48 hours** and a resolution timeline within **7 days**.

## Credentials reminder

`credentials.json` and `token.json` grant access to your Google account. Never commit them to version control. Both are listed in `.gitignore` by default. See the README for details.
